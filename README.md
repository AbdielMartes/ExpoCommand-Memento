Este repositorio contiene ejemplos de implementación de los patrones de diseño Command y Memento.
El problema que resuelve cada implementación es el siguiente:

Command:
- Da un ejemplo de procesos bancarios explicado como se resuelve el problema de
  acoplamiento fuerte y la falta de flexibilidad en las operaciones.
- Para poder ejecutar el proyecto, se necesita tener descargado typescript o
  al menos una pluggin de el mismo en Intellij Idea.
  
Memento:
- Guardar estados de una persona para su posterior uso o reuso. De forma lineal.

Los archivos memento.ts y command.ts es en donde se ejecutan y se encuentran
las implementaciones de cada patrón de diseño.

El lenguaje utilizado utilizado es el siguiente:
- TypeScript
- Creado en WebStorm 2026.1.1

Para ejecutar estos archivos desde la terminal de su Sistema Operativo, siga los siguientes pasos:
1. Ir a la página web https://nodejs.org/es/download
2. Descargar la versión compilada de node.js para el Sistema Operativo y su arquitectura que esté utilizando
3. Instalar node.js para su computadora siguiendo los pasos dados en el instalador
4. Descargar los archivos command.ts y memento.ts de la carpeta 'src' de este repositorio
5. Abrir la terminal de su Sistema Operativo
6. Ir a la ruta de ubicacion donde se descargaron los archivos memento.ts o command.ts
7. Realizar el comando 'node command.ts' o 'node memento.ts' según el archivo a ejecutar
8. Esperar a que se ejecute el archivo deseado
