Este repositorio contiene ejemplos de implementación de los patrones de diseño Command y Memento.
El problema que resuelve cada implementación es el siguiente:

Command:
- Da un ejemplo de procesos bancarios explicado como se resuelve el problema de
  acoplamiento fuerte y la falta de flexibilidad en las operaciones.
- Para poder ejecutar el proyecto, se necesita tener descargado typescript o
  al menos una pluggin de el mismo en Intellij Idea.
  
Memento:
- Guardar estados de una persona para su posterior uso o reuso. De forma lineal.

Los archivos memento.ts y command.ts es en donde se ejecutan y se encuentran
las implementaciones de cada patrón de diseño.

El lenguaje utilizado utilizado es el siguiente:
- TypeScript
- Creado en WebStorm 2026.1.1
